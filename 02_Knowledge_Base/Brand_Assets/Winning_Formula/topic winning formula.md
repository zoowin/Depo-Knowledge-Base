# topic winning formula

### 1️⃣ 核心理念（Core Philosophy）

**一句话版（可直接放 Notion）：**

> 主标题先激发好奇，不直接卖产品；
> 
> 
> 副标题承接护肤痛点；
> 
> 正文通过知识 / 测试 / 场景解释价值，顺势引入 Depology 方案。
> 

**本质逻辑：**

- 不是「产品 → 功效」
- 而是「问题 → 认知 → 方法 → 方案」

---

### 2️⃣ 内容生成公式（Topic Formula）

**标准公式：**

```
[熟悉场景 / 行为]
→[意外结果 / 被忽略的问题]
→[承诺揭示（答案 / 方法 / 测试结果）]
→[产品过渡（Depology 方案）]

```

**拆解说明（非常重要）：**

- **场景**：
    
    用户每天都在经历的时间点 / 行为
    
    > 夜间护肤、洗完脸、换季、熬夜、停用产品后
    > 
- **意外结果**：
    
    打破常识 or 揭示被忽略的风险
    
    > “你以为在修护，其实在消耗皮肤”
    > 
- **承诺揭示**：
    
    暗示“有答案”，但不一次说完
    
    > why / how / what actually works
    > 
- **产品过渡**：
    
    不是卖货，是“解决方案自然出现”
    
    > Depology = 更科学 / 更可持续 / 更适合长期使用
    > 

---

### 3️⃣ 内容原则（Content Principles）

你可以原样放进 Notion：

1. **先好奇，再卖货**
2. **隐喻资产化**（成分 = 股票 / 长期价值）
3. **缺口 + 解决方案**
4. **参与感 / 自测优先**
5. **数字化承诺**（频率、时间、结果）
6. **场景化代入**
7. **短、狠、有力（≤50 字符）**

👉 这也是你之后评估一个 topic **值不值得发**的 checklist。
