# MEL Style

**Education-First / Science-Led**

### **定位一句话**

A long-form, education-driven email style designed to explain problems, build scientific trust, and guide customers toward informed decisions.

---

### **核心目的**

**Educate → Build Trust → Convert**

---

### **适用场景**

- 成分教育（A / B / C / E 主线）
- 技术解释（如 *Micro-needling in a jar*）
- Evergreen 内容
- 品牌专业度与信任建立阶段

---

### **内容结构（固定范式）**

1. **Hero / Main Visual**
    - 产品或问题导向标题
    - 理性、清晰，不强调促销
2. **Core Problem Introduction**
    - 明确点出皮肤问题或痛点
    - 解释成因（年龄、代谢、环境等）
3. **Educational Section**
    - 科学原理 / 成分逻辑
    - 数据、百分比、研究型表达
4. **Product as Solution**
    - 产品作为“解法”出现
    - 强调 *why it works*
5. **Soft CTAs（可多次）**
    - Learn More
    - Discover More
    - Shop [Category]
6. **Brand Reinforcement**
    - Science-backed
    - Routine / long-term skincare

---

### **视觉特征**

- 纵向长图
- 白底 / 浅色背景
- 高信息密度
- 模块化段落明显
- 偏“阅读型”而非“浏览型”

---

### **语气关键词**

- Educational
- Clinical
- Informative
- Calm confidence
- Non-aggressive

---

### **不适合**

- 强促销
- 限时 / 倒计时
- 节日情绪型活动
