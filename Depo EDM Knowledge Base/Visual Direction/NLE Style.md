# NLE Style

**Conversion-First / Emotion-Driven**

### **定位一句话**

A visual-first, action-oriented email style designed to trigger emotion, create urgency, and drive immediate conversions.

---

### **核心目的**

**Trigger Emotion → Drive Action**

---

### **适用场景**

- 促销 / 折扣活动
- VIP / 限时活动
- 节日 / 季节性 Campaign
- 收益导向推送

---

### **内容结构（扁平结构）**

1. **Hero / Event Headline**
    - 情绪或事件驱动
    - 倒计时 / 错过感 / 结果导向
2. **Offer / Value Highlight**
    - 折扣
    - Bundle 价格
    - 限时信息
3. **Product Focus**
    - 产品直接出现
    - 少解释，多展示
4. **Strong CTA（可重复）**
    - Shop Now
    - Get It Now
    - Use Code

---

### **视觉特征**

- 强主视觉（人像 / 产品）
- 留白多
- 短文案
- 一屏一个信息点
- 明确价格 / 优惠对比

---

### **语气关键词**

- Urgent
- Emotional
- Direct
- Promotional
- Action-oriented

---

### **不适合**

- 深度教育
- 技术解释
- 新概念首次引入
