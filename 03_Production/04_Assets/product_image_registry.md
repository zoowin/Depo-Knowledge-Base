# Depology Product Image Registry

Store your fixed product image URLs here to reuse them across all email templates.

## How to fill in image URLs (one-time setup)
1. Click the product page link below to open it in your browser
2. Right-click the main product image → **"Copy Image Address"**
3. Paste the URL next to `Image URL:` below
4. Copy the same URL into both Klaviyo templates under `{{PRODUCT_x_IMAGE}}`

Alternatively: Upload product images once to **Klaviyo → Content → Images**, then use the Klaviyo CDN URL.

## Template Variable Quick Reference

Once you have image URLs, plug them into these variables in the Klaviyo templates:

| Variable | Templates | Description |
|---|---|---|
| `{{PRODUCT_1_IMAGE}}` | MEL + NLE | First product image URL |
| `{{PRODUCT_2_IMAGE}}` | MEL + NLE | Second product image URL |
| `{{PRODUCT_3_IMAGE}}` | MEL + NLE | Third product image URL |
| `{{HERO_IMAGE_URL}}` | MEL + NLE | Upload fresh each send |
